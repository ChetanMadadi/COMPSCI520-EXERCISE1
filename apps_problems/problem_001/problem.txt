Problem: Maximum Subarray Sum

Given an array of integers, find the contiguous subarray with the largest sum and return that sum.

Input Format:
First line: integer n (1 ≤ n ≤ 10^5) - size of array
Second line: n space-separated integers (-10^4 ≤ arr[i] ≤ 10^4)

Output Format:
A single integer representing the maximum subarray sum

Example:
Input:
5
-2 1 -3 4 -1

Output:
4

Explanation: The subarray [4] has the largest sum = 4
